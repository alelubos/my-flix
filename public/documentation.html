myFlix - API documentation

myFlix is all about the best movies of the millenium. Built with Node.js, Express and MongoDB, it allows you to get a collection of excellent movie titles. Movies can be searched by title and genre and you can access their director's Bio with curious facts. Users can sign-up and register their own account to create their personal list of 'favorites' and 'to-watch' movies.
